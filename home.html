Our Project: Blockchain-based Payment System with Fraud Detection and QR Code Payments

Our program utilises blockchain based currency, Solana, to make payments by integrating Phantom Wallet; a crypto wallet browser extension that manages your cryptocurrency portfolio. Our program also has the added benefit of providing fraud detection and prevention using Machine Learning with an estimated accuracy of 90%. With our program, all you need is a click of a button to redirect you to your Crypto account with pre-filled details of the merchant's public key and the amount, ensuring a convenient checkout experience.

(WORK IN PROGRESS) Currently, a QR code with the pre-filled information can be generated, however, integration with mobile phones is still in development.

How We Did It

Blockchain:

We integrated a reputable crypto browser extension (Phantom Wallet) into our website so that we can automatically send them a request for a certain amount of Solana. The request will contain the amount and the receiver's public key.

Machine Learning:

In order to train our model to properly be able to predict fraudulent transactions, we had to create synthetic to train on since publicly available data on websites like Kaggle keep certain important transaction data private. We simulate four months of transaction that follow a normal distribution using the Faker Library and used the SciPy Library to add randomness. Next, we used the isolation tree model to detect anomolies or outliers. When detected, that transaction is flagged as fraud.

Finally, XGBoost is used to train a model based on the synthetic data to predict the authenticity of future transactions. So far, we have achieved around 90% accuracy in detecing fraudulent transactions.
